2、MATLAB計算機與矩陣操作

2.1、Ready to launch（初步認識）

1、MATLAB Programming Modes（編程模式）

  • Command line(in command window)
  • Scripts(.m files)

2、MATLAB as A Calculator

  • Operators : + - * / ^

  • Result is computed,and displayed as ans

  • Precedence rules(優先規則):

    • Left-to-right within a precedence group
    • Precedence groups are (highest first):
      1. Parenthesis() [括號]
      2. Power(^)
      3. Multiplication and division(*,/) [乘法和除法]
      4. Addition and subtraction(+,-)
